# Mintlify Deployment Guide

Use Mintlify when documentation should deploy as a hosted docs site outside the main app.

## Preview

```bash
cd docs/mintlify
mint dev
```

## Deploy

1. Create or open the Mintlify project.
2. Connect the Git repository.
3. Set the docs root to `docs/mintlify` if the repository also contains application code.
4. Confirm the production docs URL.
5. Record the URL in the target repository's README or deployment notes.

## Verify

- `mint dev` works locally.
- The production docs URL loads.
- Navigation links resolve.
- Introduction and quickstart pages have title and description frontmatter.
