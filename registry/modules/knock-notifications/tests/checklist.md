# Knock Notifications Checklist

- [ ] module manifest validates
- [ ] Knock environment and in-app feed channel exist
- [ ] `KNOCK_API_KEY` is server-only
- [ ] public Knock env vars are scoped to preview and production
- [ ] authenticated user id is passed into the feed provider
- [ ] server workflow trigger succeeds for a test user
- [ ] read/unread feed state updates
- [ ] user preferences are respected
