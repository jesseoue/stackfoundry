# Shared Technology Skills

Shared skills live in `registry/skills/<name>/SKILL.md` and can be referenced by module manifests with `agents.skills`.

Use shared skills for technology guidance that should travel with multiple modules, such as databases, auth providers, billing providers, deployment platforms, observability tools, AI tooling, docs frameworks, and search providers. Keep module-specific operating instructions in `registry/modules/<module>/skill/SKILL.md`.
