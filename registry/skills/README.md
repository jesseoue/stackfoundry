# Shared Technology Skills

Shared skills live in `registry/skills/<name>/SKILL.md` and can be referenced by module manifests.

Use shared skills for technology guidance that should travel with multiple modules, such as databases, auth providers, billing providers, deployment platforms, observability tools, notification providers, analytics pipelines, AI tooling, AI discovery/SEO, docs frameworks, and search providers. Keep module-specific operating instructions in `registry/modules/<module>/skill/SKILL.md`.
