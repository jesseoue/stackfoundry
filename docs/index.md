1. Read the [Registry](./registry.md) docs.
2. Inspect the first modules in `registry/modules`.
3. Run `node apps/cli/src/cli.mjs validate`.
4. Try `node apps/cli/src/cli.mjs add api-keys --target /tmp/app --dry-run`.
